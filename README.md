# 輸入車販売企業「ExciteCode」（架空）のコーポレートサイト

架空の輸入車販売企業「ExciteCode」のコーポレートサイトを実装しました。  
Adobe XDのデザインカンプから静的コーディングを実施し、WordPress化まで対応しました。


## 力を入れた点
お問い合わせフォームのJavaScript実装です。  
WordPressプラグイン「Contact Form 7 Multi-Step Forms」を使うと、確認画面で他人の入力情報が表示される等の不具合が報告されていることから、プラグインを使わずに実装しました（具体的な実装機能については後述）。  
このように、リスクを考慮してプラグインに頼らない実装も、ある程度は対応可能です。


## ページ構成
- トップページ（`/`）
- 固定ページ
  - Aboutページ（`/about`）
  - Serviceページ（`/service`）
- 通常投稿
  - 一覧ページ（`/news`）
  - 個別記事ページ（`/news/slug`）
- カスタム投稿
  - 一覧ページ（`/works`）
  - 個別記事ページ（`/works/slug`）
- お問い合わせ
  - 入力ページ（`/contact`）
  - 確認ページ（`/contact`）
  - 完了ページ（`/contact/complete`）
- 404エラーページ


## デザインカンプ
`Adobe XD`


## 使用した技術
- HTML/CSS/JavaScript
- Sass（Dart Sass）
- WordPress
- PHP
- CSS設計（BEM/FLOCSS）


## 実装したWordPress機能
- トップページ（front-page.php）
- 固定ページ（page-slug.php）
- 通常投稿の一覧ページ（home.php）
- 通常投稿の個別記事ページ（single.php）
- 通常投稿の月別一覧ページ（date.php）
- 通常投稿のカテゴリー別一覧ページ（category.php）
- カスタム投稿の一覧ページ（archive-slug.php）
- カスタム投稿の個別記事ページ（single-slug.php）
- タクソノミー別一覧ページ（taxonomy.php）
- お問い合わせ入力ページ（プラグイン「Contact Form 7」を使用）
- カスタムフィールド（プラグイン「Advanced Custom Fields」を使用）
- 404エラーページ（404.php）


## 実装した機能およびUIパーツ
- ハンバーガーメニュー
- スムーススクロール
- スティッキーヘッダー
- パンくずリスト
- ページネーション
- 縦方向の画像ループスライダー（slickを使用）
- 右から左へ流れるテキスト
- テキストが一定の文字数に達したら、それ以降は3点リーダで表示
- 以下、お問い合わせ機能（JavaScriptのみで実装）
  - 必須項目がすべて入力されるまで確認画面を表示不可
  - 確認画面にユーザの入力内容を表示
  - 確認ボタンを押すと確認画面に遷移
  - 戻るボタンを押すと入力画面に戻る
  - 送信ボタンを押すと送信完了画面に遷移
